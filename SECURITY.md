# Security Policy

## Supported Versions

The active maintenance line is `0.2.x`.

## Reporting a Vulnerability

Please do not open a public issue for sensitive vulnerabilities. Email the maintainer at `997132391@qq.com` with:

- Affected version or commit
- Reproduction steps
- Expected and actual behavior
- Any proof-of-concept code, logs, or screenshots that help verify the issue

The maintainer will acknowledge confirmed reports and coordinate a fix or mitigation before public disclosure.
